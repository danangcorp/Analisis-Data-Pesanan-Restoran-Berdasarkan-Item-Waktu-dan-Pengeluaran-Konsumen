# Analisis-Data-Pesanan-Restoran-Berdasarkan-Item-Waktu-dan-Pengeluaran-Konsumen
Tujuan analisis ini adalah memahami pola pesanan restoran untuk mengetahui item yang paling banyak dan paling sedikit dipesan, pesanan dengan pengeluaran tertinggi, waktu pemesanan yang ramai dan sepi, serta menentukan jenis masakan yang perlu dikembangkan.
